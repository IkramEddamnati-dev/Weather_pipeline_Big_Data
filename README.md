# Orchestration d’un pipeline Big Data de visualisation de données
This project represents an orchestration of a Big Data pipeline for data visualization by integrating various powerful technologies that each play a specific role in the data processing and visualization workflow. 

## Technologies utilisées

- **Apache Airflow** pour l'orchestration des tâches
- **Kafka** pour le streaming de données en temps réel
- **Cassandra** pour le stockage distribué
- **Spark SQL** pour le traitement de données à grande échelle
- **Spark ML** pour l'application de modèles de machine learning
- **Streamlit** pour la visualisation interactive des données
